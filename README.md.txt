# 🌾 FarmTap – Local Farm Finder

FarmTap connects local farmers with consumers looking for fresh agricultural products.

## 🚀 Features
- Farmer profile creation
- Geo-based farm finder
- Beautiful, responsive design

## 🛠️ Technologies Used
- React + Vite
- TailwindCSS
- Firebase Authentication & Firestore
- React Router

## 📦 How to Run Locally

```bash
npm install
npm run dev
